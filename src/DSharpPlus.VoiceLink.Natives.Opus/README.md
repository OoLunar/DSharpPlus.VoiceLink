# Opus Natives

This project contains the native libraries for Opus, required by [`DSharpPlus.VoiceLink`](https://github.com/OoLunar/DSharpPlus.VoiceLink/) but may be used anywhere. These native libraries are completely unmodified from the upstream Opus project found at https://gitlab.xiph.org/xiph/opus. The libraries are built for:

- [`android-arm64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libopus/android-arm64)
- [`ios-arm64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libopus/ios-arm64)
- [`linux-arm64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libopus/linux-arm64)
- [`linux-musl-arm64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libopus/linux-musl-arm64)
- [`linux-musl-x64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libopus/linux-musl-x64)
- [`linux-x64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libopus/linux-x64)
- [`osx-arm64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libopus/osx-arm64)
- [`osx-x64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libopus/osx-x64)
- [`win-arm64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libopus/win-arm64)
- [`win-x64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libopus/win-x64)
- [`win-x86`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libopus/win-x86)
