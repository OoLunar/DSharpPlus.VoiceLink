# Sodium Natives

This project contains the native libraries for Sodium, required by [`DSharpPlus.VoiceLink`](https://github.com/OoLunar/DSharpPlus.VoiceLink/) but may be used anywhere. These native libraries are completely unmodified from the upstream Sodium project found at https://github.com/jedisct1/libsodium. The libraries are built for:

- [`android-arm64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libsodium/android-arm64)
- [`ios-arm64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libsodium/ios-arm64)
- [`linux-arm64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libsodium/linux-arm64)
- [`linux-musl-arm64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libsodium/linux-musl-arm64)
- [`linux-musl-x64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libsodium/linux-musl-x64)
- [`linux-x64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libsodium/linux-x64)
- [`osx-arm64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libsodium/osx-arm64)
- [`osx-x64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libsodium/osx-x64)
- [`win-arm64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libsodium/win-arm64)
- [`win-x64`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libsodium/win-x64)
- [`win-x86`](https://github.com/OoLunar/DSharpPlus.VoiceLink/libs/libsodium/win-x86)
